## Using Prism React Renderer

Using `prism-react-renderer`

```jsx react-live
import React from "react";

const Dump = (props) => (
  <div
    style={{
      fontSize: 20,
      border: "1px solid #efefef",
      padding: 10,
      background: "white",
    }}
  >
    {Object.entries(props).map(([key, val]) => (
      <pre key={key}>
        <strong style={{ color: "white", background: "red" }}>{key} ðŸ’©</strong>
        {JSON.stringify(val, "", " ")}
      </pre>
    ))}
  </div>
);

export default Dump;
```
